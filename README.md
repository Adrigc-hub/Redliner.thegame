# Redliner.thegame# Redliner WebXR Clone

Un clon web multipalataforma estilo Redliner con mecánicas de parkour rápidas y soporte para visores de realidad virtual (Oculus), construido con código puro usando HTML, CSS y JavaScript (Three.js).

## Características
- **Movimiento fluido:** Gravedad, aceleración e inercia.
- **Dash:** Sistema de impulso usando la tecla `Shift`.
- **Combate:** Sistema de Raycasting para disparar a los NPCs con `Click izquierdo`.
- **Inteligencia Artificial:** NPCs de práctica que persiguen al jugador.
- **WebXR:** Totalmente compatible con plataformas de realidad virtual.

## Instalación y Uso
1. Clona este repositorio o descarga los archivos.
2. No necesitas instalar ningún motor gráfico. Simplemente sirve los archivos usando un servidor local (como la extensión *Live Server* de VS Code) para que los módulos de Three.js carguen correctamente por razones de seguridad del navegador (CORS).
3. ¡Súbelo a GitHub y activa **GitHub Pages** para jugarlo en línea con tus amigos en cualquier momento!
